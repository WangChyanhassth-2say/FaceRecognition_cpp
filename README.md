# FaceRecognition_cpp
face detect, align and recognize via onnx models implemented with cpp

## Quick Strat  
### // for installing  
ensure you have cmake and makefile  
### // for compiling  
ensure you have opencv in your env  
onnx lib is provided in ./3rdparty/  
```
mkdir build  
cd build  
cmake ..  
make && ./main  
```
